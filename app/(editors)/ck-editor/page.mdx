# CK Editor

> CKEditor 5 provides every type of WYSIWYG editing solution imaginable. From editors similar to Google Docs and Medium, to Slack or Twitter-like applications, all is possible within a single editing framework. &mdash; [CKEditor 5](https://ckeditor.com/ckeditor-5/)

## Overview

### Installation

- Offers two types of build for editors: pre-built and custom build. Installation is different for each type.
  - Pre-built editors can be installed using a package manager.
  - Custom build editors can be installed by extracting the zip file and including it in the package.json file as a dependency.
- The installation process is easy and the documentation is clear and concise.
- Had a bit of problem with typesafety and rendering the custom build in production enviroment.

### Ease of use

- The editor that comes out of the box is easy to use and has a lot of features.
- Exports a React component, which can be easily integrated into a project.
- Does not come with a way to extract editor methods, so we have to use the editor instance directly.

### Pricing

- CK Editor 5 is free to use and open source under the GPL License.
- Offers a commercial license with additional features.

### Styling and customization

- The editor surface can be themed via overriding the CSS.
- Reconstruction of the toolbar is next to impossible.

### Documentation and support

- The documentation is extensive and clear.
- Can be followed here: [CKEditor 5 Documentation](https://ckeditor.com/docs/ckeditor5/latest/)

### Pros

- Offers a lot of features and is easy to use.
- Has a clear and concise documentation.
- Offers a commercial license with additional features.

### Cons

- Had a bit of problem with rendering the custom build in production enviroment.
- Does not come with a way to extract editor methods, so we have to use the editor instance directly.
- Editor surface can be themed, but the toolbar cannot be extracted/customized.
- You have to pay for additional features.
